What is Flask?

Flask is a web framework. It makes creating websites a lot easier for developers. For example, it simplifies working with urls (routing) and sending data between web pages.

What is Bootstrap?

Bootstrap is a CSS framework. It makes creating websites a lot easier for developers. For example, how professional a website looks and how easy it is to interact with the site.
